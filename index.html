Show Me Music

Show Me Music helps bridge the gap between local bands and band venues and show casing their music. We geo target a users location and then bring up a list of upcomming live shows using an API from songkick. We then build a playlist using Rdio which includes songs from these artist so the user can listen to the songs before they purchase a ticket. There is also a log in and authentication for venue owners who would like to manually add upcoming show dates. This allows venue owners a chance to change the performace schedule on the fly.
